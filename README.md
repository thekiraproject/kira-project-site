# The Kira Project

Kira is a local-first compound-AI system: one 9-billion-parameter model
serving every pipeline role — router, specialists, synthesizer, agentic
planner — with ~50 live tools, on a single Apple Silicon machine, with zero
cloud inference. It is built and operated by a single person working with AI
pair-engineering, governed by architecture decision records, and verified by
a fixture of 1,100+ live assertions that mechanically define correct behavior.

This repository publishes what we measure.

## Posts

- **[Your local AI is re-reading its own prompt](posts/2026-07-06-prefill-tax.md)**
  (July 2026) — nearly half of all model time was prefill of text the model
  had already read; how we measured it, removed it (−34% to −45% end-to-end),
  and the activation discipline that made the change trustworthy.
